Projects

One of my most significant projects has been the development of an AI speech-to-text and natural language processing pipeline that combines OpenAI Whisper, BERT, and large language model integration. This system was designed to process multilingual voice input, convert speech to text, interpret user intent, and support intelligent response workflows through a modular AI pipeline.

The project involved building the end-to-end flow from audio capture and transcription to text normalization, semantic analysis, and downstream language model interaction. By combining speech recognition with NLP-driven interpretation, the system demonstrated how multiple AI technologies can work together to support more natural and accessible human-computer interaction.

I have also worked on a major industry-focused project with Miami Valley Metal Recycling (MVMR), where I developed automation tools to support the collection, organization, and processing of operational image data. This work included building a Playwright-based data extraction pipeline to gather ticket data and associated images from ScrapRight, organize metadata, standardize file naming, and prepare datasets for scalable downstream use.

In support of that effort, I also built systems involving computer vision, wireless image transfer, cloud storage, and process automation. This included using OpenCV-based image processing, secure file packaging, AWS S3 dataset organization, and custom transmission workflows across multiple platforms. These projects have strengthened my experience in AI integration, data engineering, workflow automation, and real-world system design.
